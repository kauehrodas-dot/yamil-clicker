# Yamil Clicker

Yamil Clicker is an engaging incremental game designed for players who enjoy tapping and strategy elements.

## Overview
In Yamil Clicker, players click to earn "Yamil Points" and can purchase upgrades to increase point earnings. The goal is to amass as many points as possible!

## Features
- Simple and intuitive gameplay
- Various upgrades for point enhancement
- Achievements for milestones
- Beautiful graphics and sound effects

## Gameplay
1. **Click to Earn Points**: The player earns points with each click.
2. **Purchase Upgrades**: Use earned points to buy upgrades that boost point generation.
3. **Unlock Achievements**: Reach milestones to unlock achievements and gain bonuses.

## Installation
To run Yamil Clicker locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/kauehrodas-dot/yamil-clicker.git
   ```
2. Navigate into the directory:
   ```bash
   cd yamil-clicker
   ```
3. Open the `index.html` file in your browser.
